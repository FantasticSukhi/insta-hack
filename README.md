## INSTALLATION [Termux]

```shell
 apt update
 apt upgrade
 pkg install python
 pkg install python3
 pkg install git
 git clone https://github.com/FantasticSukhi/insta-hack
 ls
 cd insta-hack
 pip3 install -r requirements.txt
 chmod +x *
 bash setup
```

## INSTALLATION [Kali Linux]

```shell
 sudo apt install python
 sudo apt install python3
 sudo apt install git
 git clone https://github.com/FantasticSukhi/insta-hack
 ls
 cd insta-hack
 pip3 install -r requirements.txt
 chmod +x *
 sudo python3 insta-hack.py
```
